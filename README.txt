This Project is a simple User Interface Modification for playing Innovation on BoardGameArena.com

step 1:
Download the Chrome extention "Stylus"
https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne?hl=en

step 2: 
Open the extention and click "Write new style"

step 3:
Paste the entire contents of the accompanying file (script) into Stylus's new style. Save it. 


Start a game of Innovation, the modifications should take effect. You can tinker with this code to get it exactly to your liking, 
it is labeled as to exactly what changes different parts of the UI.
